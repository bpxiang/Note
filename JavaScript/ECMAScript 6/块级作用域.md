# 块级作用域
>time: 2018-02-07 17:51:36  

ES5 只有全局作用域和函数作用域，没有块级作用域。

let 实际上为 JavaScript 新增了块级作用域。
```javascript
function f1() {
    let n = 5;
    if(true) {
        let n = 10;
    }
    console.log(n); // 5
}
```

ES6 允许块级作用域任意嵌套。
```javascript
{
    {
        {
            {
                {
                    let insane = "Hello World";
                }
            }
        }
    }
}
```
