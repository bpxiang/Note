## 数组排序

可以实用 Arrays 类中的 sort 方法
```Java
int[] a = new int[10000];
...
Arrays.sort(a);
```
这个方法实用了优化的快速排序算法。快速排序算法对于大多数数据集合来说都是效率比较搞的。
